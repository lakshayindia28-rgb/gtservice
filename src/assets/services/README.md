# Service images

Place service hero images in this folder with filenames matching the service slug.

- Aspect ratio: **16:9** (landscape)
- Suggested size: **1600Ã—900** or larger
- File formats: `jpg`, `jpeg`, `png`, `webp`

The service page auto-loads the image when the filename (without extension) matches the service slug.
